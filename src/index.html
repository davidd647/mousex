MouseX

What is this?

This is MouseX - a plugin to add to your website so that there is a comfortable/fun alternative to using a mouse.

Why, though?

Using a computer mouse isn't very a11y... Also, sometimes it's just not comfortable to reach from the keyboard to the mouse every few seconds.

We use mouses for scrolling and clicking... Why not just use a rocket ship indicator controller via keyboard instead? :D

MouseX was made as an experiment for a Mintbean hackathon.

Where is Mintbean?

Mintbean is an internet-based hackathon initiative. They host hackathons every week to help developers and recruiters find each other, and encourage continual learning.

How do I use this?

For details on how to use MouseX, please check out my github repo!

Users:

  • Press cmd + x to make your cursor change into a flying saucer!
  • Press 'a', 's', 'd', 'w' keys to move the flying saucer!
  • Hold the 'q' key to stabilize your flying saucer!
  • Press the 'e' key to click the thing that's under your flying saucer!
